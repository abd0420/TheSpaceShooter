# TheSpaceShooter

A 2D space shooter built in C++ using the Raylib library. 

## What it does
You control a spaceship, dodge falling meteors, and shoot them to get a high score. The game gets faster as your score goes up, and your highest score saves automatically to a text file. 

## How to run it
1. Make sure you have a C++ compiler (MinGW) and the Raylib library installed.
2. Clone or download this repository. Make sure the `assets/` folder stays in the exact same folder as the code files.
3. Open the project folder in VS Code.
4. Press `Ctrl + Shift + B`. This uses the included `tasks.json` file to compile all the `.cpp` files together at the same time.
5. Run the `main.exe` file that generates to play the game.
